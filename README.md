# Portfolio

Personal portfolio of Bidhan Niroula — Full-stack Developer.

## Tech Stack

- **Frontend**: React, Next.js, Tailwind CSS
- **Backend**: NestJS, TypeScript
- **Database**: PostgreSQL, MongoDB
- **DevOps**: Docker, Azure

## 🚀 Getting Started

```bash
# Install dependencies
bun install

# Start dev server
bun run dev
```

## 📁 Project Structure

```
/
├── public/
│   └── assets/ # Images and logos
├── src/
│   ├── components/      # React components
│   ├── config/          # Site configuration
│   ├── lib/             # Utilities
│   ├── pages/           # Astro pages
│   └── styles/          # Global styles
└── package.json
```
